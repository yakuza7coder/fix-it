
# Fix-it

> *Status do Projeto:* Em Desenvolvimento / Fase Inicial

#Visão Geral

O **Fix-it** é um jogo em desenvolvimento utilizando a engine **GameMaker** e a linguagem **GML Code**. O foco do projeto é aplicar conceitos sólidos de lógica de programação, estruturas de dados e desenvolvimento de jogos na prática, criando uma experiência fluida de gameplay e mecânicas bem estruturadas.

---

## Funcionalidades Principais (Planejadas)

 *Movimentação Fluida:* Sistema de movimentação preciso baseado nos eixos X e Y.
 *Física de Colisão:* Sistema de colisões sólidas customizado (sem travamentos em paredes).
 *Mecânica de Reparo/Ação:* Interações dinâmicas do jogador com o ambiente do jogo.
 *Inteligência Artificial:* Problemas em empresas que sejam inteligentes
 *Interface (UI) e Atmosfera:* Barras de status na tela, efeitos sonoros e trilha sonora integrada.

---

## Estrutura do Projeto

* `Fix-it.yyp` - Arquivo principal do projeto GameMaker.
* `options/` - Configurações gerais do jogo e plataformas.
* `rooms/` - Cenários, fases e salas do jogo.
* `sprites/` - Texturas, animações e artes dos personagens/cenários *(gerada ao criar assets)*.
* `sounds/` - Efeitos sonoros e músicas de fundo *(gerada ao criar assets)*.
* `objects/` - Programação da lógica e comportamento de cada elemento do jogo *(gerada ao criar assets)*.
* `.gitignore` - Arquivo de configuração para ignorar caches temporários do GameMaker.

---

## Tecnologias Utilizadas

* **GameMaker (IDE 2026+)** - Engine de desenvolvimento.
* **GML Code (GameMaker Language)** - Linguagem de programação nativa para controle total da lógica.
* **Git & GitHub** - Controle de versão e gerenciamento de código.

---

## Pré-requisitos

* **GameMaker** instalado na máquina.
* **Git** configurado para controle de versão (opcional para desenvolvimento local).

---

## Como Executar o Projeto

1. Clone este repositório no seu computador:
   ```bash
   git clone [https://github.com/yakuza7coder/fix-it.git](https://github.com/yakuza7coder/fix-it.git)

2. Abra o GameMaker.

3. Clique em "Abrir" (Open) e navegue até a pasta clonada.

4. Selecione o arquivo Fix-it.yyp.

5. Com o projeto aberto, clique no botão Play (F5) no topo do GameMaker para rodar o jogo.
